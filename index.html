🧬 Personalise Your Agents

Each agent can be customised with two files that define who they are and how they work.

🧠SOUL.MD

Defines the agent's personality, values, and communication style. This is who they are.

📋INSTRUCTIONS.MD

Defines the agent's role, responsibilities, and operating rules. This is what they do.

Upload these via each agent's profile in the sidebar. CONTEXT.MD and TOOLS.MD are already baked in at the workspace level.

      Settings

      API Key — set a workspace-wide key here, or override it per-agent in their profile. Keys are stored encrypted in your browser.

      Model — choose Haiku 4.5 (fast & cheap), Sonnet 4.6 (balanced), or Opus 4.6 (most capable). Agents can each have their own model override.

      Cowork threshold — slide left to route more tasks to Claude Desktop (uses your Max plan); slide right to keep more on the API.

      ⚡ The Dev Agent

      Dev uses Claude Code for real coding power — it can read your project files, edit code, and run terminal commands directly. To enable this, run node server.js in your project folder before chatting.

      No server? No problem — Dev falls back to the standard API and still gives great coding advice, just without live file access.

      Backup & Restore

      Use Download Backup to save all your agents, conversations, and settings as a JSON file. Keep it somewhere safe (Google Drive works great). Use Restore from File to bring everything back on any device.
